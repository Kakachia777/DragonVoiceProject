How to Pin Dragon Voice Assistant to Taskbar

Step 1: Run the Application

Double-click on the Dragon Voice Assistant executable file to start the application.

Step 2: Find the Icon in Taskbar

When the application is running, you'll see its icon in the Windows taskbar at the bottom of the screen.

Step 3: Right-click on the Icon

Right-click on the Dragon Voice Assistant icon in the taskbar to open the context menu.

Step 4: Select "Pin to taskbar"

Click on the "Pin to taskbar" option in the menu that appears.

Step 5: Done!

The Dragon Voice Assistant is now permanently pinned to your taskbar for easy access.

You can click on the pinned icon anytime to start the application, even when it's not running.

Note: If you have trouble starting the application, try the following:
  • Use the "(Admin)" shortcut if available
  • Make sure all files stay together in the same folder
  • Check that your microphone is properly connected
